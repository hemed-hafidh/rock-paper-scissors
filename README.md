# rock-paper-scissors

This is a project from [The Odin Project] (https://www.theodinproject.com/lessons/foundations-rock-paper-scissors) curriculum.

## Goal

The goal of this project is to create a simple rock-paper-scissors game using Javascript with no gui interface. The game run on console developer tool. It is played between human and random guess provided by the computer, it runs in five round and then stop.
